CVA Submission Expectations

Accepting Student Work

Linked work should not be accepted because it may cause access issues, permission problems, or uncertainty about what was submitted.

Core Rules

All Work Must Be Submitted in CTLS

CVA student work must be submitted inside CTLS Coursework. Files should be uploaded directly to the assignment, not shared as links to OneDrive, Google Drive, Canva, or another outside platform.

Do Not Accept

Links to OneDrive, Google Drive, Canva, shared folders, online documents, slide decks, videos, or outside websites.

Why It Matters

Linked Work Cannot Be Verified

Linked work can create access issues, permission problems, and confusion about what was actually submitted. Students may also be able to edit linked files after the due date.

If the teacher has to leave CTLS or request permission to view the work, the assignment has not been submitted correctly.

Teacher Action

What Teachers Should Do

If a student submits work incorrectly, message the student, explain what needs to be fixed, and reopen the assignment so they can resubmit correctly. Give the student a clear deadline to correctly submit the assignment and let them know the assignment will be marked Missing if it is not corrected by the deadline.

Suggested Teacher Message

"Hi there, I noticed that your assignment was not submitted correctly. CVA assignments must be uploaded directly in CTLS Coursework, not shared as a OneDrive, Google Drive, Canva, or other external link. Please upload the actual file using My Attachments by this Friday once I reopen the assignment for you."

Important Note

Text Editor Uploads

When students upload a file directly in their text editor, rather than from the My Attachments button, their submission appears to be a link.

However, unlike a linked file, which can still be edited by the student after submission, this is an unchangeable file. Even so, students should be directed to resubmit correctly using the My Attachments button.
