CVA Submission Expectations

Accepting Student Work in CVA Classes

Use this guide to help students submit work correctly inside CTLS Coursework.

Important

There Are No Exceptions to These Rules

All student work in CVA classes must be submitted inside CTLS Coursework. This helps teachers access, review, provide feedback, and document student submissions in the correct location.

Rule 1

All work must be submitted inside CTLS Coursework.

Rule 2

All files and documents must be uploaded rather than linked to a location outside of CTLS, such as OneDrive or Canva.

Clarification

What Counts as a Link?

We have noticed some confusion about what constitutes a link. A link takes you outside of CTLS to a site such as OneDrive or Canva to access the student's work.

If a student submits a link to a file outside of CTLS, the work may still be editable after submission. For that reason, linked work cannot be accepted as a valid CVA submission.

Important Note

Important Note About Text Editor Uploads

When students upload a file directly in their text editor, rather than from the My Attachments button, their submission appears to be a link.

However, unlike a linked file, which can still be edited by the student after submission, this is an unchangeable file. Even so, students should be directed to resubmit correctly using the My Attachments button.

Action Steps

What Teachers Should Do

Step 1

Check the Submission

Look carefully to determine whether the student submitted a link, a text editor upload, or a file attached with My Attachments.

Step 2

Message the Student

Use a clear, supportive message to explain what needs to be corrected.

Step 3

Reopen the Assignment

Once the student confirms they understand, reopen the assignment so they can resubmit correctly.
