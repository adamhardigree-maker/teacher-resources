Navigating CTLS

Class Schedule

Every CVA course has a Class Schedule that students can access inside their CTLS classroom. It outlines the expected timeline for completing course content and is one of the most useful reference points a teacher has for monitoring pacing and making outreach specific.

Overview

Your Pacing Reference Point

When you open the Progress Tracker and see a student who has not submitted recently, the Class Schedule helps you determine how serious that gap actually is. Without it, you are estimating. With it, you can see exactly where the student should be relative to a published course timeline.

Students have access to the Class Schedule inside their CTLS classroom. That means when you reference it in outreach or announcements, you are pointing them to something they can already see.

Big Idea: Teachers do not set or modify the Class Schedule, but referencing it before outreach makes your communication more targeted and more useful to students.

Three Ways to Use It

How Teachers Use the Class Schedule

Use the Class Schedule as a reference point before communicating with students, families, or local school contacts about pacing and progress.

Use 1

Before Student Outreach

Check the Class Schedule before you send a message to a student who appears behind. It tells you which assignments or units the student should have completed by now, so your outreach can name specifics rather than speaking in generalities.

Teacher move: Instead of saying “please catch up,” say, “You are missing Unit 3 work, which was due last week according to your Class Schedule.”

Use 2

In Class Announcements

When you post a pacing reminder to all students, reference the Class Schedule by name. Directing students to something they can actually find is more useful than a general reminder to stay on track.

Teacher move: Say, “Check your Class Schedule to make sure you are on track with this week’s assignments.”

Use 3

In Conversations with Students and Parents

When a student or parent asks about a student’s progress, the Class Schedule gives you a shared reference point. You can speak to where the student should be in the course and where they actually are.

Teacher move: Use the schedule to describe pacing clearly instead of relying only on the current course average.

Why It Matters

Why This Matters

Anchors Outreach

Knowing where a student should be makes your intervention timely and specific rather than reactive and general.

Student-Facing

Students already have access to it, so referencing it reinforces a tool they should be using on their own.

Supports Collaboration

It gives teachers and local school staff a common language when discussing student progress and next steps.

Quick Takeaways

What to Remember

✓

The Class Schedule is published inside every student’s CTLS classroom.

✓

Teachers do not set or modify it, but it should inform pacing conversations.

✓

Reference it by name in announcements and outreach so students know where to look.

✓

Use it before contacting local schools so you can speak specifically about student progress.
